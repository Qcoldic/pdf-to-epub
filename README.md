# Кодекс — PDF → EPUB конвертер

Клиентское веб-приложение для конвертации PDF-файлов в формат EPUB. Работает полностью в браузере — файлы не отправляются на сервер.

## Два режима конвертации

### 📄 Точная копия страниц (по умолчанию)
Каждая страница PDF рендерится в JPEG и встраивается в EPUB с фиксированной вёрсткой (`rendition:layout pre-paginated`). **Сохраняются все картинки, формулы, таблицы и точная вёрстка** — результат выглядит как оригинальный PDF. Текст при этом не выделяется (растровое изображение).

### 📝 Текст (перетекаемый)
Из PDF извлекается текстовый слой, абзацы и «красные строки» восстанавливаются с учётом отступов в исходнике. Подходит для художественной литературы и документов с простой вёрсткой. **Картинки и точная вёрстка не сохраняются** — для них используйте режим «Точная копия».

## Как это работает

1. Откройте `index.html` в браузере (локально через `file://` или на GitHub Pages).
2. Перетащите PDF-файл в зону загрузки или выберите через кнопку.
3. Отредактируйте название, автора и язык книги при необходимости.
4. Выберите режим: «Точная копия страниц» (картинки + вёрстка) или «Текст (перетекаемый)» (лёгкий текст).
5. Нажмите «Собрать EPUB» — конвертер обработает страницы и соберёт валидный EPUB 3.
6. Готовый файл скачается автоматически.

## Технологии

- **Vanilla HTML/CSS/JS** — без фреймворков и сборщиков
- **pdf.js** (v3.11.174) — рендеринг и извлечение текста из PDF
- **JSZip** (v3.10.1) — сборка ZIP/EPUB-архива
- **Google Fonts** — Fraunces (заголовки) + Inter (интерфейс)

Внешние библиотеки загружаются с jsDelivr CDN. Сама конвертация — 100% локальная.

## Формат выходного файла

- EPUB 3 с NCX-фоллбэком для совместимости со старыми читалками
- Одна XHTML-страница на каждую страницу PDF
- Содержание (nav.xhtml + toc.ncx)
- Валидный `content.opf` с метаданными (название, автор, язык, дата)
- **Точная копия:** фиксированная вёрстка, JPEG-изображения страниц, viewport-метатеги
- **Текст:** перетекаемый текст, CSS с учётом абзацных отступов (`p.indent`)

## Ограничения

- **Сканированные PDF** (изображения без текстового слоя) — в текстовом режиме текст не извлекается. Используйте режим «Точная копия страниц» или предварительно прогоните OCR.
- **PDF с паролем** — необходимо сначала снять защиту.
- Сложная вёрстка (таблицы, колонки, формулы) точно воспроизводится только в режиме «Точная копия».
- Работает в современных браузерах (Chrome, Firefox, Safari, Edge).
