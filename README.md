# Кодекс — PDF → EPUB конвертер

Клиентское веб-приложение для конвертации PDF-файлов в формат EPUB. Работает полностью в браузере — файлы не отправляются на сервер.

## Как это работает

1. Откройте `index.html` в браузере (локально через `file://` или на GitHub Pages).
2. Перетащите PDF-файл в зону загрузки или выберите через кнопку.
3. Отредактируйте название, автора и язык книги при необходимости.
4. Нажмите «Собрать EPUB» — конвертер извлечёт текст постранично и соберёт валидный EPUB 3.
5. Готовый файл скачается автоматически.

## Технологии

- **Vanilla HTML/CSS/JS** — без фреймворков и сборщиков
- **pdf.js** (v3.11.174) — извлечение текста из PDF
- **JSZip** (v3.10.1) — сборка ZIP/EPUB-архива
- **Google Fonts** — Fraunces (заголовки) + Inter (интерфейс)

Внешние библиотеки загружаются с jsDelivr CDN. Сама конвертация — 100% локальная.

## Формат выходного файла

- EPUB 3 с NCX-фоллбэком для совместимости со старыми читалками
- Одна XHTML-страница на каждую страницу PDF
- Содержание (nav.xhtml + toc.ncx)
- Валидный `content.opf` с метаданными (название, автор, язык, дата)

## Ограничения

- **Сканированные PDF** (изображения без текстового слоя) — текст не извлекается. Для таких файлов нужен OCR, который здесь не поддерживается.
- **PDF с паролем** — необходимо сначала снять защиту.
- Изображения из PDF не переносятся — извлекается только текст.
- Сложная вёрстка (таблицы, колонки, формулы) может быть воспроизведена неточно.
- Работает в современных браузерах (Chrome, Firefox, Safari, Edge).
